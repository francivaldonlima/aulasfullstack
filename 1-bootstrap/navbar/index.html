Front-end

Aprenda a criar interfaces de usuário com HTML, CSS e JavaScript. São as tecnologias que rodam no navegador e que permitem que o seu site ou aplicativo funcione. Você vai aprender a criar páginas web, a entender como elas funcionam e a criar interfaces de usuário interativas.

Back-end

Aprenda a criar servidores e APIs com Node.js, Python, Ruby e outras linguagens. São as tecnologias que rodam no servidor e que permitem que o seu site ou aplicativo funcione. Você vai aprender a criar um servidor, a entender como ele funciona e a criar APIs para que outras pessoas possam usar os dados que você disponibiliza.
